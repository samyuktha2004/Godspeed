# UPGRADE POLICY
#   - Do NOT upgrade langgraph beyond 0.2.x without auditing agent/graph.py.
#     The 1.x series has breaking StateGraph API changes.
#   - Do NOT upgrade langchain-core to 1.x while langgraph is on 0.2.x.
#   - Do NOT upgrade langchain-openai to 1.x — it requires langchain-core>=1.3.2
#     which conflicts with langgraph 0.2.x. Keep on 0.3.x.
#   - Do NOT upgrade langchain-google-genai beyond 2.x while langchain-core
#     is on 0.3.x. The 3.x/4.x series requires langchain-core >=1.0.
#   - Do NOT upgrade neo4j to 6.x — the async driver session API changed.
#   - torch version must remain compatible with numpy <2.0 until numpy is
#     explicitly upgraded and tested with FlagEmbedding.
# =============================================================================
